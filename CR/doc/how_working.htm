Описание логики приложения
1.Это приложение написано для наглядной демонстрации и работы с аvl-деревом поиска
Для подробностей формата описания бинарного дерева перейдите по гиперсылке на домашней странице
2. Для выбора файла с данными - нажмите на кнопку Choise file и после выберите файл
в диалоговом окне вашей файловой системы. Формат файла должен быть txt.
3. После обработки файла программа выведет текущее состояние бианрного дерева. Его можно менять,
добавляя или удаляя узлы:
-) Для добавления необходимо ввести имя нового узла в поле справа от кнопки Add node и затем нажать
на нее
-) Для удаления необходимо ввести имя удаляемого узла в поле справа от кнопки Delete node и затем
нажать на нее
