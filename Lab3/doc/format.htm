Описания формата данных

В файле символьная строка должна иметь вид: aDbDcDd..., где каждая строка а, b, с, d,...,
в свою очередь, имеет вид х1 С х2. Здесь x1 есть строка, состоящая из символов А и В, а х2 - строка,
обратная строке x1 (т. е. если x1 = АВАВВ, то х2 - ВВАВА). Таким образом, исходная строка состоит только
из символов А, В, С и D

Пример


AABCBAADABCBA
