Описание логики приложения
1.Это приложение проверяет строку на соответствие с видом: aDbDcDd...
Для подробностей перейдите по гиперсылке на домашней странице
2. Для выбора файла с данными - нажмите на кнопку Choise file и после выберите файл
в диалоговом окне вашей файловой системы. Формат файла должен быть txt.
3. После обработки файла проверьте поле Result.
