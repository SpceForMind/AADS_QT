Описание логики приложения
1.Это приложение вычисляет высоту деерва
Для подробностей формата описания бинарного дерева перейдите по гиперсылке на домашней странице
2. Для выбора файла с данными - нажмите на кнопку Choise file и после выберите файл
в диалоговом окне вашей файловой системы. Формат файла должен быть txt.
3. После обработки файла проверьте поле Height.
Значение -1 означает, что дерево пусто, другие значения означает, что дерево не пусто и имеет высоту. При неправильной расстановке скобок при описании дерева, будет выведено сообщение "Incorrect data from file"
